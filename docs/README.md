# Documentation

## Quick Start

```bash
npx vibemon
```

The desktop app launches in the system tray and listens on `http://127.0.0.1:19280`.

For the ESP32 hardware display, see [vibemon-esp32](https://github.com/opspresso/vibemon-esp32).
